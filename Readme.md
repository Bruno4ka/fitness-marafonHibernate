# Система управления фитнес-марафонами (Hibernate)

Курсовая работа по дисциплине "Технологии хранения и обработки данных"

---

## Описание проекта

Консольное приложение для управления системой фитнес-марафонов. Реализовано на Java с использованием Hibernate (ORM) для взаимодействия с PostgreSQL.

### Основные возможности:

- CRUD операции для всех сущностей через GenericRepository
- Сложные бизнес-запросы на JPQL
- Транзакционная обработка данных
- Пул соединений HikariCP
- Логирование через SLF4J + Logback

---

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Язык | Java 17 |
| Система сборки | Maven |
| СУБД | PostgreSQL 17 |
| ORM | Hibernate 7.0.0.Final |
| Спецификация | JPA 3.2 |
| Пул соединений | HikariCP |
| Логирование | SLF4J + Logback |

## Структура базы данных

База данных содержит 7 таблиц:

| Таблица | Описание |
